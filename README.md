<b>方式</b>
・送受信処理はJQuery Ajaxをベースに
・Ajaxイベント処理はサーバサイトJavascript
・サーバプログラムはファンクションチェン
・Ajaxイベントのデータ収集を自動化
・Ajaxイベントのデータ表示を自動化
・Ajaxイベントの単純入力チェックを自動化
・バッチプログラムは画面と同じ方式
<b>機能</b>
・ファイルアップロードダウンロード機能
・ショットカットキー機能
・メール送信機能
・WEB画面からPDFへの変換機能
・ルールエンジン連携機能
・corsサポート
・・・
<b>制限</b>
・JDK1.7 あるいは JDK1.8。
  ※JDK1.6は入力チェックが正しく動作しない。
・JQuery1 あるいは JQuery2。
  ※JQuery UIのため、JQuery3を利用できない。
</pre>
